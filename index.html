Apa itu website statis?
Website statis adalah website yang kontennya tidak berubah secara dinamis. Halaman-halamannya ditulis dalam HTML dan ditampilkan kepada setiap pengunjung dengan cara yang sama. Website statis lebih cepat, aman, dan mudah di-hosting.
Bagaimana cara embed video YouTube?
Untuk embed video YouTube, Anda perlu menggunakan tag iframe dengan URL embed YouTube. Ambil ID video dari URL YouTube (bagian setelah "v="), lalu masukkan ke dalam format: https://www.youtube.com/embed/VIDEO_ID. Pastikan untuk mengatur atribut allow dan allowfullscreen agar video dapat diputar dengan baik.
Apakah website ini responsif?
Ya, website ini dibuat dengan desain responsif menggunakan CSS modern. Ini berarti tampilan akan menyesuaikan dengan ukuran layar perangkat yang digunakan, baik itu desktop, tablet, maupun smartphone. Kami menggunakan teknik seperti flexbox, media queries, dan unit relatif untuk memastikan pengalaman yang optimal di semua perangkat.
Bagaimana cara mengubah konten website ini?
Untuk mengubah konten, Anda cukup mengedit file HTML ini. Untuk video, ganti ID video di bagian src iframe. Untuk FAQ, tambahkan atau ubah div dengan class "faq-item" sesuai kebutuhan. Semua styling sudah diatur di bagian CSS, jadi Anda tidak perlu file terpisah.
Apakah saya perlu server khusus untuk hosting?
Tidak, karena ini adalah website statis, Anda bisa meng-hostingnya di layanan gratis seperti GitHub Pages, Netlify, Vercel, atau Cloudflare Pages. Anda juga bisa menggunakan web hosting tradisional. Tidak diperlukan database atau server-side processing, sehingga biaya hosting sangat minimal atau bahkan gratis.
